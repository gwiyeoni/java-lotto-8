# java-lotto-precourse
# 로또


## 구현할 기능 목록

- [x] 로또 구입
  - [x] 로또 구입 금액 입력받기
    - [x] 예외 : 숫자가 아닌 문자를 입력 받았을 경우
    - [x] 예외 : 0 이상의 숫자가 아닐 경우
    - [x] 예외 : 1000으로 나누어 떨어지지 않을 경우
  - [x] 구매한 로또 개수 출력하기
  - [x] 로또 번호 생성하기
  - [x] 로또 번호 오름차순으로 정렬하기
  - [x] 로또 번호 출력하기
- [ ] 당첨 판별
  - [x] 당첨 번호 6개를 쉼표를 기준으로 입력받기
    - [x] 예외 : 쉼표를 기준으로 6개가 아닌 경우
    - [x] 예외 : 6개 모두 숫자가 아닌 경우
    - [x] 예외 : 1~45 범위 안에 있지 않은 경우
    - [x] 예외 : 중복이 있는 경우
  - [x] 보너스 번호 1개를 입력받기
    - [x] 예외 : 숫자가 아닌 경우
    - [x] 예외 : 1~45 범위 안에 있지 않은 경우
    - [x] 예외 : 당첨 번호와 중복되는 경우
  - [ ] 당첨 기준 판별하기
  - [ ] 당첨 내역 계산하기
  - [ ] 총 수익률 게산하기
- [ ] 결과 출력
  - [ ] 당첨 내역 출력
  - [ ] 총 수익률 출력
- [ ] 예외 처리
  - [ ] 잘못된 값을 입력할 경우 [ERROR]로 시작하는 에러메시지 출력 후 그 부분부터 입력 다시 받기
